tent-client-java
================

Java Tent protocol client

## Installation

Add the following repository to your POM's `<repositories>` tag:

````xml
<repository>
  <id>sonatype-nexus-snapshots</id>
  <url>https://oss.sonatype.org/content/groups/public/</url>
  <snapshots>
    <enabled>true</enabled>
  </snapshots>
</repository>
````
